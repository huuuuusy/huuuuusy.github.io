Easy-Question 561-Array Partition I

Question

Given an array of 2n integers, your task is to group these integers into n pairs of integer, say (a1, b1), (a2, b2), …, (an, bn) which makes sum of min(ai, bi) for all i from 1 to n as large as possible.

Examples

Input: [1,4,3,2]

Output: 4
Explanation: n is 2, and the maximum sum of pairs is 4 = min(1, 2) + min(3, 4).

Note

n is a positive integer, which is in the range of [1, 10000].

All the integers in the array will be in the range of [-10000, 10000].

Alogrithm

In order to understand this approach, let us look at the problem from a different perspective. We need to form the pairings of the array’s elements such that the overall sum of the minimum out of such pairings is maximum. Thus, we can look at the operation of choosing the minimum out of the pairing, say (a, b)(a,b) as incurring a loss of a - ba−b(if a> ba>b), in the maximum sum possible.

The total sum will now be maximum if the overall loss incurred from such pairings is minimized. This minimization of loss in every pairing is possible only if the numbers chosen for the pairings lie closer to each other than to the other elements of the array.

Taking this into consideration, we can sort the elements of the given array and form the pairings of the elements directly in the sorted order. This will lead to the pairings of elements with minimum difference between them leading to the maximization of the required sum.

Solution

Python3

class Solution:
def arrayPairSum(self, nums):
    """
       :type nums: List[int]
       :rtype: int
       """
       nums.sort()
       return sum(nums[::2])

notes

  1. sort() method on a list sorts that list into ascending order, e.g. list.sort().
  2. nums[A:B:C] means from the Ath number to Bth number in the list, choose one number ever C numbers.Thus, sum(nums[::2]) means add the first, third, fifth…… number of the list.

Java

class Solution {
    public int arrayPairSum(int[] nums) {
        Arrays.sort(nums);
        int sum = 0;
        for (int i = 0; i < nums.length; i += 2){
             sum += nums[i];
        }
        return sum;
    }
}

notes

  1. Array.sort() method is a java.util.Arrays class method.

Easy-Question 1-Two Sum

Question

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Examples

Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

Alogrithm

Solution

Python3

class Solution:
def twoSum(self, nums, target):        
    d = {}
    for index, item in enumerate(nums):
        minus = target - item
        if minus in d:
            return [d[minus], index]
        else:
            d[item] = index

notes

  1. enumerate is a function that return the indexs and items of a list.
  2. d={} is a dictionary to store the index and item of the original list, and it is also very convenient to search the value by index.

Java

class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();
        for (int i = 0; i< nums.length; i++){
            int minus = target - nums[i];
            if (map.containsKey(minus)){
                return new int[]{map.get(minus),i};
            }
            map.put(nums[i],i);
        }
        throw new IllegalArgumentException("No two sum solution");
    }
}

notes

  1. HashMap() likes the dictionary in Python, which is used to store the key and value.
  2. map.containsKey() is used to search the value by key.
  3. map.get() is used to get the value by key.
  4. map.put() is used to add a new element in the HashMap.
  5. throw new IllegalArgumentException() is used to show the error for the wrong answer.
