This post is about some basic conceptions of mulit-layer neural network.

Multi-layer Perceptron

Concept

We use matrix to store the weights. Each row in the matrix corresponds to the weights sent from the same input node, and each column corresponds to teh weight of the same hidden node. For the figure above, the weight matrix equals:

We can use Numpyt to initialize the weight matrix. 

# Number of records and input units
n_records, n_inputs = features.shape

# Number of hidden units
n_hidden = 2

#Initialize
weights_input_to_hidden = np.random.normal(0, n_inputs**-0.5, size=(n_inputs, n_hidden))

Next, we need to calculate the hidden layer input.

We can use np.dot in Numpy to calculate the product of input layer matrix and the input weight matrix.

hidden_inputs = np.dot(inputs, weights_input_to_hidden)

Example

Calculating the forward-propagating of a neural network with 4 input units, 3 hidden units and 2 output units.

Package Import

import numpy as np

Define Sigmoid Function

def sigmoid(x):
    return 1/(1+np.exp(-x))

Define Network Size

N_input = 4
N_hidden = 3
N_output = 2

Data Initialization

np.random.seed(42)
# Make some fake data
X = np.random.randn(4)

weights_input_to_hidden = np.random.normal(0, scale=0.1, size=(N_input, N_hidden))
weights_hidden_to_output = np.random.normal(0, scale=0.1, size=(N_hidden, N_output))

Calculate the Forward-propagating

hidden_layer_in = np.dot(X, weights_input_to_hidden)
hidden_layer_out = sigmodi(hidden_layer_in)

out_layer_in = np.dot(hidden_layer_out,weights_hidden_to_output)
output_layer_out = sigmoid(hidden_layer_in)

Back Propagation

Concept

We can calculate the hidden layer error by using the output layer error as follow:

Then update the weight by using:

In fact, this calculation can be used in any layer. For weight between each layer, the update weight matrix euqals the product of learning rate, error of the layer output and the value of layer input.

Example

Package Import

import numpy as np

Define Sigmoid Function

def sigmoid(x):
    return 1/(1+np.exp(-x))

Define Parameters

x = np.array([0.5, 0.1, -0.2])
target = 0.6
learnrate = 0.5

weights_input_hidden = np.array([[0.5, -0.6],
                                 [0.1, -0.2],
                                 [0.1, 0.7]])

weights_hidden_output = np.array([0.1, -0.3])

Forward Pass

hidden_layer_input = np.dot(x, weights_input_hidden)
hidden_layer_output = sigmoid(hidden_layer_input)

output_layer_in = np.dot(hidden_layer_output, weights_hidden_output)
output = sigmoid(output_layer_in)

Backwards Pass

Calculate Output Error

error = target - output

Calculate Error Term for Output Layer

output_error_term = error * output * (1 - output)

Calcualte Error Term for Hidden Layer

hidden_error__term = np.dot(output_error_term, weights_hidden_output) * hidden_layer_output * (1 - hidden_layer_output)

Calculate Change in Weights for Hidden Layer to Output Layer

delta_w_h_o = learnrate * output_error_term * hidden_layer_output

Calculate change in weights for input layer to hidden layer

delta_w_i_h = learnrate * hidden_error_term * x[:, None]
