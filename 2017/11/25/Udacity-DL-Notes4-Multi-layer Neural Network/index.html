多层感知器

概念

我们使用矩阵来存储权重。 矩阵中的每一行对应于从相同输入节点发送的权重，并且每列对应于相同隐藏节点的权重。 对于上图，权重矩阵等于：

我们可以使用Numpyt初始化权重矩阵。

# Number of records and input units
n_records, n_inputs = features.shape

# Number of hidden units
n_hidden = 2

#Initialize
weights_input_to_hidden = np.random.normal(0, n_inputs**-0.5, size=(n_inputs, n_hidden))

接下来，我们需要计算隐藏层输入。

我们可以在Numpy中使用np.dot来计算输入层矩阵和输入权重矩阵的乘积。

hidden_inputs = np.dot(inputs, weights_input_to_hidden)

后向传播

概念

我们可以通过使用输出层错误来计算隐藏层错误，如下所示：

更新权重：

实际上，此计算可用于任何层。 对于每层之间的权重，更新权重矩阵表示学习速率、层输出的误差和层输入的值的乘积。

例子

导入包

import numpy as np

定义Sigmoid

def sigmoid(x):
    return 1/(1+np.exp(-x))

定义参数

x = np.array([0.5, 0.1, -0.2])
target = 0.6
learnrate = 0.5

weights_input_hidden = np.array([[0.5, -0.6],
                                 [0.1, -0.2],
                                 [0.1, 0.7]])

weights_hidden_output = np.array([0.1, -0.3])

前向传播

hidden_layer_input = np.dot(x, weights_input_hidden)
hidden_layer_output = sigmoid(hidden_layer_input)

output_layer_in = np.dot(hidden_layer_output, weights_hidden_output)
output = sigmoid(output_layer_in)

后向传播

计算输出误差

error = target - output

计算输出层的Error Term

output_error_term = error * output * (1 - output)

计算隐藏层的Error Term

hidden_error__term = np.dot(output_error_term, weights_hidden_output) * hidden_layer_output * (1 - hidden_layer_output)

计算隐藏层到输出层的权重变化

delta_w_h_o = learnrate * output_error_term * hidden_layer_output

计算输入层到隐藏层的权重变化

delta_w_i_h = learnrate * hidden_error_term * x[:, None]
